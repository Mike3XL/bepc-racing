# BEPC Racing Analytics

A system for Ballard Elks Paddle Club (BEPC) members to view race results, series standings, and handicap calculations.

See [SPEC.md](SPEC.md) for full project specification.

## Project Structure

```
bepc-racing/
├── SPEC.md              # Project specification
├── core/                # Core Java library (handicap engine, data model)
├── cli/                 # Command-line tool
├── api/                 # Lambda API functions
├── web/                 # Static website
└── infra/               # AWS CDK infrastructure
```
